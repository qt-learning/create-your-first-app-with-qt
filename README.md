# Creating Your First App with Qt Design Studio

This is the example project done in "Creating Your First App with Qt Design Studio" course.

> Get the full learning experience by enrolling in the course on [Qt Academy](https://www.qt.io/academy/course-catalog#creating-your-first-app-with-qt-design-studio)


## 🎯 What You'll Learn

- How to create an application prototype with Qt Design Studio?
- What are Qt Quick Controls?
- What components exist within Qt Quick Controls to create an application?
- What is anchoring?
- What are models?
- How to organize content with models?
- How to add functionality with additional code to an app?

⭐ Skill Level: Beginner

⏱️ Time to Complete: 60 minutes

## Instructions

To get started with this project, you will need to have Qt Design Studio installed on your computer. You can download Qt Design Studio from the [Qt.io website](qt.io).

Once you have Qt and Qt Design Studio installed, head over to [Qt Academy](academy.qt.io) and follow along to create the project template and add the assets to the project.
